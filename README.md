# bookmarks.retromultiplayer.com

Personal web browser bookmark collection

## Extensions to replace "New Tab" content

- Microsoft Edge: [Custom New Tab URL](https://microsoftedge.microsoft.com/addons/detail/custom-new-tab-url/oeibmbobgpgnbnlbaffdgebpeepfbnhi)
- Google Chrome: [Simple New Tab URL](https://chromewebstore.google.com/detail/simple-new-tab-url/pkklffjkfmhpoiepobbldnadjflciinb)
- Mozilla Firefox: [New Tab Override](https://addons.mozilla.org/en-US/firefox/addon/new-tab-override/) (a bit buggy)
- Vivaldi: no extension required, set in Core Settings

Vivaldi and Floorp
also allow using this kind of bookmark page
as custom sidebar

## Development environment

Install requirements:
`pip install -r requirements.txt`

Build output from templates:
`python compile.py`

Run a HTTP server of your choice, e.g. [Five Server for Visual Studio Code](https://marketplace.visualstudio.com/items?itemName=yandeu.five-server)
